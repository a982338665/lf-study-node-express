# node-express
express

**1.express-generator生成器：**

    1.可以快速生成express框架
    2.安装生成器(全局)：npm/cnpm i -g express-generator
    3.初始化及使用：
        1.命令生成初始化项目：
            express -e 项目名称 （目录即为名称）
            express -e （将当前目录作为express初始化项目）
        2.安装依赖：
             install dependencies:
                 > npm install
             run the app:
                 > SET DEBUG=node-express:* & npm start
